## How to install
To run the site locally, follow the next steps:
1. Clone the repository:
 ```sh
   git clone https://github.com/danielafp106/KTT-First-Task.git
   ```
2. Add the dependecies:
```sh
   npm install
   ```
3. Run the web app:
```sh
   npm start
   ```
